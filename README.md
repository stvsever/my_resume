# Stijn Van Severen - resume website

This is the repository that hosts my personal resume website, served with GitHub Pages.

It is a single static site (no build step):

- `index.html` - structure and content
- `styles.css` - dark, responsive design system
- `script.js` - brain-network background (Yeo 7-network layout), manuscript radar, and UI interactions
- `assets/` - portrait and manuscript title pages
- `files/` - downloadable resume PDF

Live site: stijnvanseveren.com
